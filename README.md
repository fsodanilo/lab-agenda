# lab-agenda

Projeto inicial para a disciplina de Engenharia de Software.

## Descricao

Este repositório foi preparado para o desenvolvimento de uma aplicacao Python relacionada ao laboratorio de agenda.

No estado atual, o projeto contem a configuracao inicial de versionamento e esta pronto para receber o codigo-fonte.

## Requisitos

- Python 3.10 ou superior
- Git

## Como comecar

1. Clone o repositorio.
2. Crie um ambiente virtual:

```bash
python3 -m venv .venv
```

3. Ative o ambiente virtual:

```bash
source .venv/bin/activate
```

4. Instale as dependencias do projeto quando elas forem adicionadas.
